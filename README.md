**Repo for my personal website. Visit [subtlebits.com](https://subtlebits.com/) to have a look.**

Built using React, Next.js and styled-components.

---

**Setup**

Clone:

```bash
git clone git@github.com:dinohamzic/www-subtlebits-com.git
```

Install:

```bash
npm install
```

Run:

```bash
npm run dev
```
